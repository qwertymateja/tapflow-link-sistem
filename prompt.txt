Add these new features to the dashboard and public page:

1. CUSTOM GRADIENT BUILDER (Design → Background → Gradient):
- Keep the 8 preset gradients
- Add a "Custom" option below presets
- When Custom is selected, show:
  - Color picker for gradient start color
  - Color picker for gradient end color
  - Direction selector (Top to Bottom, Left to Right, Diagonal ↘, Diagonal ↙)
- Live preview updates in real time
- Save custom gradient to profiles table

2. TEXT SIZE & BOLD OPTIONS (Design → Text):
- Display name size slider (small / medium / large / xl)
- Bio text size slider (small / medium / large)
- Display name bold toggle (on/off)
- Bio bold toggle (on/off)
- All update live preview in real time

3. NEW LINK TYPES (Dashboard → Links → Add):
When adding a link, user can choose link type:

TYPE 1 - Regular Link (existing):
- Title + URL → opens URL on click

TYPE 2 - Text Block:
- Title (shown as button)
- Content (text area, supports line breaks)
- When clicked on public page → opens a modal/overlay with the text content
- Close button on modal

TYPE 3 - Image Block:
- Title (shown as button)  
- Image upload or URL
- When clicked on public page → opens a modal with the image displayed
- Close button on modal

TYPE 4 - WiFi Card:
- Title (default "WiFi" but editable)
- Network name (SSID) field
- Password field (shown as dots, but copyable)
- QR code image upload (optional)
- When clicked on public page → opens a modal showing:
  - Network name with copy button
  - Password with show/hide toggle and copy button
  - QR code image if uploaded (tap and hold to save on mobile)
  - "Connect" button that tries to connect via: wifi://SSID:PASSWORD (works on some Android devices)
- Copy buttons show "Copied!" feedback for 2 seconds
