In the WiFi modal on the public page, the action buttons (Copy Password, etc.) 
should use the user's custom button colors instead of hardcoded purple.
Apply the same button style, button color, button text color, and border radius 
that the user has set in Design → Buttons.
This applies to all modals (WiFi, Text, Image, PDF) — all buttons inside 
modals should match the user's custom button design.
